# Threat Modeling Lab

## What I’m learning
How to identify threats, vulnerabilities, and risks in a system.

---

## System Example
Retail website storing customer data.

---

## Threats

- Hacker steals customer data
- Insider accesses data illegally
- Weak password attack

---

## Vulnerabilities

- No MFA
- Weak password policy
- Unencrypted database

---

## Risks

- Data breach
- Financial loss
- Legal penalties

---

## Mitigations

- MFA
- Encryption
- Access control

---

## Skills learned
- Threat identification
- Risk analysis
- Security thinking
